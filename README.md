>  基于 ionic3 构建的天气信息查询APP

### 项目技术架构
***
*  ionic3
*  angular4

### 安装步骤
***
``` bash
# 安装依赖
$ npm install

# 添加平台
$ ionic cordova platform add android

# 运行
$ ionic cordova run android --livereload

# 打包
$ ionic cordova build android --ant--release
```
